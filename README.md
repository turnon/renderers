# Renderers

To know how this render is invoked

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'renderers'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install renderers

## Usage

Put this line in template (maybe erb file), then render:

```ruby
<%= Renderers %>
```
